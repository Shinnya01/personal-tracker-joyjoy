# AGENTS.md

## Project
Personal offline-first Tracker PWA.

## Stack
- Vue 3
- TypeScript
- Vite
- Tailwind CSS v4
- Pinia
- Vue Router
- Dexie.js
- vite-plugin-pwa

## Rules
- One user only.
- No login.
- No backend.
- No API.
- Store all tracker data in IndexedDB using Dexie.
- Store images in IndexedDB, not localStorage.
- App must work offline.
- Mobile-first iPhone-style UI.
- Keep code clean, typed, and component-based.

## Commands
- Install: npm install
- Dev: npm run dev
- Build: npm run build
